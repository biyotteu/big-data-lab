### 1. 디렉토리 구조

```
BigDataLab/
|-- _config.yml          # 사이트 설정 파일
|-- _layouts/            # 레이아웃 파일
|   |-- default.html     # 기본 레이아웃
|-- _includes/           # 재사용 가능한 HTML 코드 (헤더, 푸터 등)
|   |-- header.html      # 헤더
|   |-- footer.html      # 푸터
|-- _data/               # 데이터 파일 (연구 프로젝트 목록 등)
|-- _posts/              # 블로그 게시물 (연구 발표 글 등)
|-- assets/              # CSS, JS, 이미지 파일
|   |-- css/             # 스타일 파일
|   |-- js/              # 스크립트 파일
|   |-- images/          # 이미지 파일
|-- index.md             # 메인 페이지
|-- about.md             # 소개 페이지
|-- research.md          # 연구 섹션
|-- _sass/               # SCSS 파셜 파일 모음
|   |-- _variables.scss  # 변수 설정 (색상, 폰트 등)
|   |-- _mixins.scss     # 믹스인 설정 (반복되는 스타일 함수)
|   |-- _base.scss       # 기본 스타일 (reset 등)
|   |-- _header.scss     # 헤더 스타일
|   |-- _footer.scss     # 푸터 스타일
|   |-- _home.scss       # 메인 페이지 스타일
|   |-- _components.scss # 버튼, 카드 등 컴포넌트 스타일
|   |-- _responsive.scss # 반응형 스타일
```
